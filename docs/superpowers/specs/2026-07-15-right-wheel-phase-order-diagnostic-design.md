# 右轮中间两相交换诊断设计

## 现状

左右两只裸露输出轴在独立单轮测试中都只轻微抖动。两侧 ULN2003 输入灯按序变化，零步侧保持断电，四组电机绕组均约 24 欧姆，电机红色公共线与驱动板 VCC 导通，5V 轨从空载 5.25V 降至带载最低 4.9V。STM32 到驱动板当前均按 `IN1、IN2、IN3、IN4` 直连。

这些结果排除了轮毂卡滞、单个电机、单块驱动板、明显供电不足、绕组开路和公共线错位，下一步验证电机线圈物理顺序是否需要交换中间两相。

## 临时接线

只在电机 5V 断开时调整右侧 ULN2003 输入：

| STM32 | 当前 | 临时验证 |
| --- | --- | --- |
| PB0 | IN1 | IN1，不变 |
| PB1 | IN2 | IN3 |
| PB10 | IN3 | IN2 |
| PB11 | IN4 | IN4，不变 |

左侧 ULN2003、两侧 VCC/GND 和两个电机五线插头保持不变。换线后先确认导线牢固、没有裸线短路、两板指示灯全灭，再接回 5V。

## 固定动作

使用一次性 USB 探针复用正式 `DeviceService`，只发送一次 `MOVE_WHEELS(0, +64, 120, 300, 1500)`。探针沿用现有连接、ESTOP、故障、活动命令和结果校验，不接受外部动作参数，不自动重试。

执行时左侧 ULN2003 必须全程不亮，右侧按临时相序换相。结束、拒绝、超时或异常均释放两侧输出；异常时尝试 STOP 并保留 ESTOP。

## 结果与后续

- 若右侧裸轴连续转过小而可见的角度，判定该批电机需要 `IN1、IN3、IN2、IN4` 的逻辑相位映射。立即断开 5V，随后恢复右侧直连接线；永久修复放在 STM32 板级 ULN2003 映射层，使外部接线继续保持 IN1-IN4 直连。
- 若右侧仍只抖动或不动，立即断开 5V 并恢复 `PB1→IN2、PB10→IN3`，不修改固件相序，转入逐相输出电压测量。
- 无论结果如何，本次不测试左轮、不执行反转、不增加速度、步数或持续时间。
