# 右轮中间两相交换诊断实施计划

## Task 1：断电换线门禁

操作员断开电机 5V，只将右侧 ULN2003 的 `PB1` 从 IN2 移到 IN3、`PB10` 从 IN3 移到 IN2。PB0、PB11、左侧驱动板、VCC/GND 和电机五线插头保持不变。确认导线牢固、无短路后接回 5V，并再次确认裸轴、两板灯全灭和可立即断电。

## Task 2：准备一次性探针

临时创建固定右轮探针，复用正式 `DeviceService`，参数为左 0 步、右 +64 半步、120 half-steps/s、300 half-steps/s² 和 1500 ms 超时。主机语法检查通过后才允许运行。

## Task 3：执行与观察

探针只恢复 USB 切换造成的链路丢失 ESTOP，确认无活动命令和不可恢复故障后进入 MANUAL，发送唯一动作。操作员观察右侧裸轴是否连续转动；左板必须全程不亮，动作结束后两板必须全灭。任何异常均停止且不重试。

## Task 4：恢复

恢复 ESP32 正式程序并删除探针。测试后立即断开电机 5V。若右轴转动，恢复右侧 IN1-IN4 直连接线，进入 STM32 板级相位映射修复；若仍抖动，也恢复原接线并转入逐相输出电压检查。
